What It Does

Zero-Trust Analysis

Treats every email as a graph of atomic assertions. Does not trust the visual rendering of the client.

Franken-Thread Detection

Identifies when a sender has edited the quoted history of a previous email (inline interjections).

Visual Fingerprinting

Extracts CSS styles (Red, Bold, Uppercase) to detect efficient aggression.

Offline-First

Runs locally using SQLite and lightweight NLP (Spacy), avoiding cloud dependencies for privacy.
